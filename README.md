# 🚗 Real-Time Vehicle Detection System

A real-time vehicle detection and tracking application built using YOLO and Streamlit. This project allows users to upload images or videos and perform object detection directly in the browser.

---

## 🔍 Features

* Real-time vehicle detection using YOLO
* Image and video upload support
* Interactive Streamlit interface
* Visualization of detection results
* Training analytics (confusion matrix, dataset distribution)

---

## 🧠 Technologies Used

* Python
* Ultralytics YOLO
* OpenCV
* Streamlit

---

## 🚀 Demo

Upload an image or video and the model will detect vehicles in real time.

---
